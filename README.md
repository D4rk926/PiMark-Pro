# PiMark-Pro
Test your Raspberry Pi useing PiMark Pro and see how good is your cooling, cpu, ram, and more!

How to download?
-
1 Download the zip

2 Open the terminal

3 Paste this code in: bash -c "$(curl -sL https://raw.githubusercontent.com/D4rk926/PiMark-Pro/main/install_auto.sh)"

4 Delet the auto_installer.sh

5 Enjoy it

How to use it?
-
1. Open the app from your desktop

2. Press "Enter test lab"

3. Start a test or a banchmark

4. Make food on your cpu over time
